# RickAndMortyInfoApp


Türkçe
- Proje Rick & Morty çizgi dizisinin açık API aracılığı ile uygulamaya dökülmesi üzerine
kurgulanmıştır.


Açılan ilk ekranda Rick and Morty dizisinin bölümleri API aracılığıyla alınarak flatlist içerisinde görüntülenmiştir.
Ekrandaki herhangi bir bölüme tıklandığında ilgili bölümün detaylarıyla ilgili sayfaya kullanıcı yönlendirilir.
Bölüm detaylarının bulunduğu sayfadaki bölümde oynayan karakterlere basılarak karakterin ayrıntılarını görüntüleyebilirsiniz. 

Projedeki sayfalar screens klasörünün içerisinde yer almaktadır.Bu sayfaların yönlendirmesinde kullanılan React Navigation teknolojisi navigation klasörünün içerisindedir.
Uygulamadaki arayüzlerde kullanılan stiller styles.js dosyası üzerinden çağrılmaktadır.
